This section covers the essential commands and workflows for developing, building, and maintaining the Windows98 app. All commands should be run from the app workspace directory.

## Start development server
```bash
pnpm dev
```

## Build for production 
```bash 
pnpm build
```

## Preview production build
```bash
pnpm preview
```

## Run linting
```bash
pnpm lint:code:run
pnpm lint:styles:run
pnpm lint:types:run
```
