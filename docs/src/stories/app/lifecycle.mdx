This section details the complete application startup sequence, from initial bootstrap to the full desktop environment. Understanding this flow is essential for debugging and extending the app.

## **Bootstrap** (`main.tsx` â†’ `bootstrap-app.tsx`):
   - Initializes React application
   - Sets up internationalization (i18n)
   - Renders the MainContainer

## **Container Initialization** (`MainContainer`):
   - Executes `useInitAppConfig` hook
   - Configures browser environment, language, theme
   - Renders HomeView and ProgramsPortal

## **View Composition** (`HomeView`):
   - Combines PulpitLayout (desktop background)
   - Integrates NavigationLayout (taskbar)

## **Program Management** (`ProgramsPortal`):
   - Lazy-loads programs on demand
   - Manages program lifecycle
   - Provides program isolation and performance optimization
