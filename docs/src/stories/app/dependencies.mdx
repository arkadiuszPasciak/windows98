The Windows98 app leverages a comprehensive ecosystem of workspace packages to deliver the complete desktop experience. Each dependency serves a specific purpose in the overall architecture, from UI components to business logic services.

## **@windows98/micro-frontends**:
- All interactive applications and UI components
## **@windows98/micro-services**:
- Business logic and data management services  
## **@windows98/design-system**:
- UI components, styles, and theming
## **@windows98/i18n**:
- Internationalization configuration
## **@windows98/configs**:
- Shared configuration utilities
## **@windows98/toolkit**:
- Common TypeScript utilities and types
