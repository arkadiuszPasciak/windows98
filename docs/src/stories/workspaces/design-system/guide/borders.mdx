:root[theme="light"] {

## Border Guidelines

### Do's ‚úÖ

1. **Use border tokens** for consistent 3D effects
   ```css
   box-shadow: var(--ds-border-raised);
   ```

2. **Choose appropriate border type** for component state
   - `raised`: Default buttons, frames, containers
   - `sunken`: Input fields, pressed buttons, panels
   - `field`: Specifically for form inputs

3. **Maintain Windows 98 authenticity** with proper layered effects

4. **Consider interaction states** with appropriate border changes

### Don'ts ‚ùå

1. **Don't use traditional borders** that break the 3D effect
   ```css
   /* ‚ùå Bad */
   border: 1px solid #ccc;
   
   /* ‚úÖ Good */
   box-shadow: var(--ds-border-raised);
   ```

2. **Don't mix border styles** inconsistently within components

3. **Don't ignore theme differences** - borders change between light/dark themes

## Border Radius Usage

### Component Border Radius

```css
/* Small radius for subtle rounding */
.subtle-rounded {
  border-radius: var(--ds-border-radius-sm);
}

/* Medium radius for containers */
.container-rounded {
  border-radius: var(--ds-border-radius-md);
}

/* Large radius for special elements */
.large-rounded {
  border-radius: var(--ds-border-radius-lg);
}

/* Circular elements */
.circular {
  border-radius: var(--ds-border-radius-round);
}

/* Sharp corners (authentic Windows 98) */
.sharp {
  border-radius: var(--ds-border-radius-none);
}
```

## Custom Border Extensions

### Project-Specific Borders

```css
:root {
  /* Custom border effects */
  --ds-border-focus: var(--ds-border-field), 0 0 0 1px var(--ds-color-blue300);
  --ds-border-error: var(--ds-border-field), 0 0 0 1px var(--ds-color-red100);
  --ds-border-success: var(--ds-border-field), 0 0 0 1px var(--ds-color-custom-green);
}
```

### State-Specific Borders

```css
/* Interactive states */
.ds-input.error {
  box-shadow: var(--ds-border-error);
}

.ds-input.success {
  box-shadow: var(--ds-border-success);
}

.ds-button.focus-visible {
  box-shadow: var(--ds-border-raised), var(--ds-border-focus);
}
```
