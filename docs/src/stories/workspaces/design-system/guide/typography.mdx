Typography system and font usage for the Windows98 design system.

## Font Families

The Windows98 design system uses authentic MS Sans Serif fonts to recreate the classic Windows 98 experience.

```css
/* Font Family Tokens */
--ds-font-sans-serif-bold: "MSSansSerifBold", sans-serif;
--ds-font-sans-serif-medium: "MSSansSerifMedium", sans-serif;
```

## Typography Guidelines

### Do's ✅

1. **Use typography tokens** for consistent text styling
   ```css
   font-family: var(--ds-font-sans-serif-medium);
   font-size: var(--ds-font-size-xs);
   ```

2. **Maintain hierarchy** with appropriate font sizes and weights
   - Bold fonts for headings and emphasis
   - Medium fonts for body text and UI elements

3. **Consider readability** with proper line heights and letter spacing

4. **Use semantic HTML** elements with typography classes
   ```html
   <h1 class="ds-heading-1">Page Title</h1>
   <p class="ds-text-body">Body content</p>
   ```

### Don'ts ❌

1. **Don't use system fonts** that break the Windows 98 aesthetic
   ```css
   /* ❌ Bad */
   font-family: Arial, sans-serif;
   
   /* ✅ Good */
   font-family: var(--ds-font-sans-serif-medium);
   ```

2. **Don't use arbitrary font sizes** outside the type scale

3. **Don't ignore accessibility** - ensure sufficient contrast and readable sizes

## Font Loading

### Font Face Configuration

```css
/* Ensure fonts are loaded properly */
@font-face {
  font-family: "MSSansSerifMedium";
  src: url("/fonts/MSSansSerifMedium.woff2") format("woff2");
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: "MSSansSerifBold";
  src: url("/fonts/MSSansSerifBold.woff2") format("woff2");
  font-weight: 800;
  font-style: normal;
  font-display: swap;
}
```

### Font Preloading

<!-- Preload critical fonts for better performance -->

/>
