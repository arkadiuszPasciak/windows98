## Overview

The `@windows98/design-system` package provides a comprehensive collection of React components, styles, and design tokens that recreate the authentic Windows 98 user interface. It serves as the foundation for building nostalgic Windows 98 applications with modern web technologies.

## Why?

The design system exists to solve several key challenges in UI development:

- **Consistency**: Ensures all components follow authentic Windows 98 design patterns
- **Reusability**: Provides a library of pre-built, tested components
- **Theming**: Supports both light and dark Windows 98 themes
- **Accessibility**: Modern accessibility features within retro aesthetics
- **Type Safety**: Full TypeScript support for enhanced developer experience
- **Testing**: Comprehensive unit and UI testing for reliability
