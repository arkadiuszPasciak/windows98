## What is MS App Config?

MS App Config is a centralized configuration management service that handles application settings, preferences, and configuration data across the Windows98 environment. It provides a unified way to store, retrieve, and manage configuration values for all applications and services.

## Integration Example

```tsx
import { MSAppConfig } from '@windows98/micro-services'

const appConfig = new MSAppConfig()

// Initialize configuration
await appConfig.initialize({
  environment: 'production',
  configPath: './config',
  validation: true
})

// Get configuration values
const themeConfig = appConfig.get('ui.theme', 'classic')
const apiEndpoint = appConfig.get('api.endpoint')

// Set configuration values
appConfig.set('user.preferences.language', 'en-US')
appConfig.set('app.debugMode', false)

// Listen for configuration changes
appConfig.onChange('ui.theme', (newTheme) => {
  console.log(`Theme changed to: ${newTheme}`)
})
```
