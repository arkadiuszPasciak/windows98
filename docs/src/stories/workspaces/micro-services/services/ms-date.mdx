## What is MS Date?

MS Date is a comprehensive date and time management service that provides utilities for date manipulation, formatting, timezone handling, and calendar operations. It ensures consistent date handling across all Windows98 applications with proper localization and timezone support.

## Integration Example

```tsx
import { MSDate } from '@windows98/micro-services'

const dateService = new MSDate()

// Get current date and time
const now = dateService.now()
const currentDate = dateService.getCurrentDate()

// Format dates
const formatted = dateService.format(now, 'MM/dd/yyyy HH:mm:ss')
const localized = dateService.formatLocalized(now, 'full', 'en-US')

// Date calculations
const tomorrow = dateService.addDays(now, 1)
const weekAgo = dateService.subtractDays(now, 7)
const daysDiff = dateService.daysBetween(date1, date2)

// Timezone operations
const utcTime = dateService.toUTC(now)
const localTime = dateService.toTimezone(utcTime, 'America/New_York')

// Relative time
const timeAgo = dateService.timeAgo(lastModified) // "2 hours ago"
const timeUntil = dateService.timeUntil(deadline) // "in 3 days"
```
