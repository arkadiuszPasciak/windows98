## What is MS Browser Env?

MS Browser Env is a service that provides comprehensive browser environment detection, capability assessment, and compatibility management. It ensures that Windows98 applications work consistently across different browsers and platforms by providing standardized environment information and polyfills.

## Integration Example

```tsx
import { MSBrowserEnv } from '@windows98/micro-services'

const browserEnv = new MSBrowserEnv()

// Detect browser environment
const envInfo = browserEnv.getEnvironmentInfo()
console.log(`Browser: ${envInfo.browser} ${envInfo.version}`)
console.log(`OS: ${envInfo.os}`)

// Check feature support
const hasWebGL = browserEnv.hasFeature('webgl')
const hasLocalStorage = browserEnv.hasFeature('localStorage')

// Get device capabilities
const deviceInfo = browserEnv.getDeviceInfo()
console.log(`Device type: ${deviceInfo.type}`)
console.log(`Screen: ${deviceInfo.screen.width}x${deviceInfo.screen.height}`)

// Load polyfills if needed
if (!browserEnv.hasFeature('fetch')) {
  await browserEnv.loadPolyfill('fetch')
}
```
