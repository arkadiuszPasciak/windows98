## What is MS Language?

MS Language is a comprehensive internationalization (i18n) and localization (l10n) service that manages language settings, translations, and cultural formatting across the Windows98 environment. It provides seamless multi-language support with dynamic language switching and cultural adaptation.

## Integration Example

```tsx
import { MSLanguage } from '@windows98/micro-services'

const languageService = new MSLanguage()

// Initialize with language
await languageService.initialize({
  defaultLanguage: 'en-US',
  supportedLanguages: ['en-US', 'es-ES', 'fr-FR', 'de-DE'],
  fallbackLanguage: 'en-US'
})

// Basic translation
const welcome = languageService.t('welcome.message') // "Welcome to Windows98"
const greeting = languageService.t('greeting', { name: 'John' }) // "Hello, John!"

// Pluralization
const itemCount = languageService.t('items.count', { count: 5 })
// "5 items" or "5 elementos" based on current language

// Change language
await languageService.setLanguage('es-ES')

// Format numbers and dates
const price = languageService.formatCurrency(99.99, 'USD') // "$99.99" or "99,99 ‚Ç¨"
const date = languageService.formatDate(new Date()) // Localized date format

// React integration
export const useTranslation = () => {
  const [currentLanguage, setCurrentLanguage] = useState('en-US')
  
  const t = useCallback((key, params) => {
    return languageService.t(key, params)
  }, [currentLanguage])
  
  return { t, setLanguage: languageService.setLanguage }
}
```
