## What is MS File Manager?

MS File Manager is a comprehensive file system service that provides file operations, directory management, and file system navigation capabilities. It offers a unified interface for file handling that works consistently across different environments while maintaining the classic Windows98 file management experience.

## Integration Example

```tsx
import { MSFileManager } from '@windows98/micro-services'

const fileManager = new MSFileManager()

// File operations
const fileContent = await fileManager.readFile('/documents/readme.txt')
await fileManager.writeFile('/documents/output.txt', 'Hello World')
await fileManager.copyFile('/source.txt', '/backup/source.txt')

// Directory operations
const files = await fileManager.listDirectory('/documents')
await fileManager.createDirectory('/projects/new-project')
const dirSize = await fileManager.getDirectorySize('/documents')

// File search
const searchResults = await fileManager.searchFiles({
  query: '*.txt',
  directory: '/documents',
  recursive: true
})

// File watching
fileManager.watchFile('/config.json', (event, filename) => {
  console.log(`File ${filename} ${event}`)
})

// File metadata
const fileInfo = await fileManager.getFileInfo('/documents/data.json')
console.log(`Size: ${fileInfo.size}, Modified: ${fileInfo.lastModified}`)

// Storage information
const storageInfo = await fileManager.getStorageInfo()
console.log(`Free space: ${storageInfo.freeSpace}`)
```
