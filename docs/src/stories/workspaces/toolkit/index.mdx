## Why?

The philosophy behind the `@windows98/toolkit` package is to provide a curated set of tools and utilities for efficient software development within the Windows98 ecosystem. This package contains reusable helpers, types, and utilities that enhance productivity and maintain consistency across all workspaces.

## Key Features

- **Design Patterns** - Implementation of common design patterns (Builder, Observer, Singleton Factory)
- **Type Safety** - Comprehensive TypeScript type definitions for enhanced development experience
- **Utility Functions** - Collection of commonly used utility functions for everyday development tasks
- **Zero Dependencies** - Lightweight package with no external dependencies
- **Tree Shakable** - Modular exports for optimal bundle size
