The Windows98 app provides a comprehensive desktop experience with the following key features:

## **Desktop Environment**
- **Authentic Windows 98 UI**: Faithful recreation of the classic Windows 98 interface including taskbar, start menu, and desktop icons
- **Multi-window Management**: Support for multiple simultaneous applications with proper window management
- **System Tray**: Functional system tray with clock, audio controls, and system information

## **Built-in Applications**
The Windows98 app comes with a comprehensive suite of applications that recreate the classic Windows 98 experience:

### **Productivity & Utilities**
- **Notepad**: Text editor for creating and editing documents
- **Calculator**: Full-featured calculator for mathematical operations
- **Timer**: Countdown timer utility for time management
- **File Runner**: Application launcher and file execution tool
- **Save File**: File saving and management utility
- **System Information**: Display system details and hardware information

### **System & Controls**
- **Settings**: System configuration and customization options
- **Theme Manager**: Visual theme and color scheme management
- **Language Manager**: Multi-language support and localization settings
- **Audio Button**: System audio controls and volume management
- **Clock**: System clock display with time/date information
- **Timezone**: Timezone selection and world clock functionality
- **Navigation**: System navigation and window management
- **Pulpit**: Desktop environment and icon management
- **Calendar**: Date picker and calendar functionality

### **Entertainment**
- **Tic-Tac-Toe**: Classic strategy game implementation

### **System Actions**
- **Shutdown**: System shutdown and restart options
- **Start Menu**: Application launcher and system menu

*All applications are built as micro-frontends in `@windows98/micro-frontends` for modular architecture and independent deployment.*

## **Theming & Customization**
- **Theme Manager**: Support for multiple visual themes and color schemes
- **Language Manager**: Internationalization support with multiple language options
- **Audio Controls**: System audio management and controls

## **Modern Architecture**
- **Micro-frontend Architecture**: Modular design with independent, loadable components
- **State Management**: Reactive state management using MobX
- **Type Safety**: Full TypeScript implementation for enhanced developer experience
- **Hot Reloading**: Fast development with Vite's hot module replacement
