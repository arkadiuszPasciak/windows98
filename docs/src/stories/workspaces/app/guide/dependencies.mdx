The Windows98 app is built with carefully selected modern dependencies that provide robust functionality while maintaining performance:

## **Core Runtime Dependencies**
- **React** - Latest React version with modern concurrent features
- **React DOM** - DOM rendering library for React applications
- **MobX** - Reactive state management library for predictable state updates
- **MobX React Lite** - Lightweight React bindings for MobX
- **i18next** - Internationalization framework for multi-language support
- **React i18next** - React integration for i18next

## **Workspace Dependencies**
- **@windows98/configs** - Shared configuration files and constants
- **@windows98/i18n** - Internationalization resources and translations
- **@windows98/design-system** - UI components and styling system
- **@windows98/micro-frontends** - Individual application components (Calculator, Notepad, etc.)
- **@windows98/micro-services** - Backend services and utilities

## **Development Dependencies**
- **TypeScript** - Type checking and enhanced developer experience
- **Vite** - Fast build tool and development server
- **@vitejs/plugin-react** - React support for Vite
- **Biome** - Fast linter and formatter for code quality
- **Stylelint** - CSS/SCSS linting and formatting
- **@windows98/toolkit** - Development utilities and shared tools
