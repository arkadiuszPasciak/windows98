Complete guide to using TypeScript configurations from `@windows98/configs` for type checking and compilation.

## Overview

TypeScript configurations provide type checking and compilation settings optimized for different environments and frameworks within the Windows98 ecosystem.

## Install Dependencies

```bash
# Install typescript
pnpm add -D typescript

# Install the configs package
pnpm add -D @windows98/configs
```

## Base Configuration

### Basic Usage

```json
{
  "extends": "@windows98/configs/typescript/base",
}
```

### Configuration Details

```json
{
  "extends": "@windows98/configs/typescript/base",
  "compilerOptions": {
    "allowImportingTsExtensions": true,
    "allowJs": true
  },
  "exclude": ["test123"]
}
```
